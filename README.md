
# Build Instructions
## Windows
### Prerequisite
* Visual Studio 2017
* premake5

### Building
1. Install premake5 and generate for vs2017
2. Build the visual studio solution

## Linux
### Prerequisite
* gcc or clang
* premake5

### Building
1. generate makefiles by using the "gmake" target for premake5
2. execute "make" to build all targets
